# 🛒 Smart E-commerce AI Assistant

AI-powered smart shopping co-pilot for Indian consumers.

## 🚀 Features
- 🔍 Product Search with Full AI Analysis
- ⚖️ Compare Products Side by Side
- 📊 Review Analytics Dashboard
- 🕵️ Fake Review Detection (8-point AI analysis)
- 📉 Price Drop Prediction
- 🌿 Sustainability Score
- 👗 AI Outfit Generator
- 🚨 Return Fraud Detection

## 🛠️ Tech Stack
- Python, Streamlit
- TextBlob (NLP Sentiment Analysis)
- Plotly (Data Visualization)
- Rule-based AI Engine

## ⚡ How to Run
pip install streamlit requests pandas plotly textblob
python -m textblob.download_corpora
streamlit run app.py
